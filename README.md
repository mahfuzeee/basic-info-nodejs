# basic-info-nodejs
This repository is for creating a Basic informational site using basic Nodejs. No express or other framework. 
Used Node Modules:
>>http
>>path
>>fs

page according to the url.
localhost:8080 >> take users to index.html
localhost:8080/about >> take users to about.html
localhost:8080/contact-me >> take users to contact-me.html
404.html >> display any time the user tries to go to a page not listed above.
